# 8 Years, One Journey

A single-page farewell letter chronicling **8 years at John Deere** — from Graduate Engineer Trainee to Lead Engineer II → BRT (Blue River Technology).

## What Is This?

A personal, beautifully styled HTML page written as a goodbye note to the team. It walks through five career chapters, lessons learned, and a closing letter of gratitude.

## How to View

Open `index.html` in any browser. No build step, no dependencies — just a single self-contained HTML file with inline CSS and Google Fonts.
